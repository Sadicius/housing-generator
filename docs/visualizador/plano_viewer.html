Visor de planos

Carga el JSON generado por el CLI (--output layout.json, o los varios edificio_planta_*.json de --import-seleccion) y dibuja la planta real, no solo las coordenadas. Todo se procesa en el propio navegador -- ningún archivo se sube a ningún sitio.
Selecciona uno o varios JSON (uno por planta) -- se detecta automáticamente por metadata/nombre de archivo.
Carga un archivo para ver la planta generada.
