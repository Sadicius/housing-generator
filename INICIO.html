Los enlaces de documentación son archivos .md -- tu navegador puede ofrecer descargarlos en vez de mostrarlos, según su configuración; ábrelos con cualquier editor de texto o visor de Markdown si es así. El generador (relaciones_espaciales.html) se abre directamente, sin instalar nada ni necesitar conexión salvo para la generación real en el navegador (Pyodide, vía CDN).
¿Quieres usar el CLI de Python en vez del dashboard (automatización, desarrollo)? Ejecuta instalar.sh (Mac/Linux) o instalar.bat (Windows) -- no hace falta para el dashboard, solo para esto.
